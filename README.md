# Trabalho-Sallum
Formatação:  
-Cada pasta deve ser para apenas uma página (& vice-versa)  
-Todos os arquivos de .css, .js, e .html devem ter o nomes de suas pastas parentes  
-Arquivos adicionais como imagens e audio devem ficar em pastas separadas dentro da pasta de codigo(por favor sigam algum sistema logico ao nomear os arquivos)
-Arquivos devem seguir formatação kebab case (kebab-case), variáveis, ids & classes devem usar camel case (camelCase)
