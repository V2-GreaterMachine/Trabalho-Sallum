# Trabalho-Sallum
Formatação:
-Cada pasta deve ser para apenas uma página (& vice-versa)
-Todos os arquivos de .css, .js, e .html devem ter o nomes de suas pastas parentes
-Arquivos adicionais como imagens e audio devem ficar em pastas separadas dentro da pasta em que está sendo usada (ex: pasta "img" para arquivos de imagem)
-Arquivos devem seguir formatação kebab case (kebab-case), variáveis, ids & classedevem usar camel case (camelCase)
